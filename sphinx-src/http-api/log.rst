Log
###

Methods
*******

GET
===

Returns the log from the controller.

``GET /api/log``

Returns a JSON object with the following attributes:

.. list-table::
   :widths: 3 3 10
   :header-rows: 1

   * - Attribute
     - Value Type
     - Description
   * - ``log``
     - string
     - The whole log from the controller
